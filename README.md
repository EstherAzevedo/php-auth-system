# PHP Auth System

Sistema completo de autenticação desenvolvido com:

- PHP 8
- MySQL
- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Cadastro de usuário
- Login
- Logout
- Sessão
- Dashboard protegida
- Alteração de senha
- Hash de senha com password_hash

## Banco de Dados

Importe o arquivo:

database/schema.sql

## Executar

1. Configurar Apache
2. Configurar MySQL
3. Atualizar config/database.php
4. Acessar localhost/php-auth-system
